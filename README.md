# logic-db

## Examples

To view the examples,  
clone the logic-db repo and install the dependencies :

```bash
$ git clone git://github.com/xieyuheng/logic-db
$ cd logic-db
$ npm install
```

Run example :

```bash
$ node examples/sicp/boston-high-tech-company.js
```
